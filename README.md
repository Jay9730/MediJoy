🏥 MediJoy – Healthcare Web Application
📌 Overview

MediJoy is a simple and user-friendly frontend healthcare web application that allows users to book medical appointments through a web interface.
The project is built using core web technologies and demonstrates client-side form handling and email integration without a backend.

🚀 Features
📝 Appointment Booking Form
📧 Email Notification using FormSubmit
⚡ Instant Form Validation using JavaScript
💻 Fully Frontend (No Backend Required)
🎯 Simple and Clean UI
🛠️ Tech Stack
HTML – Structure of the application
CSS – Styling and layout
JavaScript – Logic and interactivity
⚙️ How It Works
User fills out the appointment form
Clicks on “Book Appointment”
Form data is sent to FormSubmit
FormSubmit forwards the data to the configured email address
Appointment details are received via email
📧 Email Integration

This project uses FormSubmit to send emails directly from the frontend.

<form action="https://formsubmit.co/your-email@gmail.com" method="POST">
No backend required
Email received from: submissions@formsubmit.co
Includes all user input details
